## MedSAM2 inference using DDPT mask guided Prompt (Point)
1. In `MedSAM2_Inference_DDPT-Guided.py`, update the directory path and folder name to match the dataset structure within the `/DATA/Dataset_name/` directory.
2. Execute the following command to save the masks, guided by DDPT masks:
   ```bash
   python MedSAM2_Inference_DDPT-Guided.py
   ```
   
This repository is based on the official [Medical-SAM2 repository](https://github.com/SuperMedIntel/Medical-SAM2), with some additional files.
